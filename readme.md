# Music Dynamic Music Video Generator

## 项目介绍
本项目是一个Adobe After Effects自动化脚本工具，用于批量批量生成带有动态滚动字幕的音乐视频。通过简单的配置和素材准备，即可快速生成高质量的音乐视频合成，支持根据歌词文件自动生成同步的动态字幕效果。

## 功能特点
- 批量处理多个音乐文件，自动生成对应视频
- 支持动态滚动字幕效果，与音乐同步
- 可自定义字幕样式、动画效果和布局
- 灵活的素材管理，支持多种文件格式
- 根据是否包含歌词歌词文件自动切换两种不同的模板布局

## 支持的文件格式
- **封面/碟面/背景**：所有图片格式及MP4视频格式
- **歌词**：LRC格式
- **音乐**：WAV、MP3格式

## 安装与使用

### 环境要求
- Adobe After Effects (支持ExtendScript的版本)
- ExtendScript Toolkit 或 VS Code (可选，用于脚本编辑和运行)

### 基本使用流程

1. **准备工作**
   - 下载并解压项目文件到本地目录
   - 确保Adobe After Effects已正确安装

2. **运行脚本**
   - 在Adobe After Effects中运行：执行`start.jsx`文件
   - 在外部编辑器中运行：在ExtendScript Toolkit或VS Code中执行`gui.jsx`文件

3. **配置项目**
   - 在弹出的GUI界面中，指定一个空目录作为工作目录
   - 点击"创建目录"按钮生成标准素材目录结构
   - 在生成的目录中放置相应的素材文件：
     - 背景图层
     - 封面图层
     - 碟片图层
     - 音乐文件
     - 歌词文件（可选）

4. **自定义设置**
   - 在"基本设置"标签页配置路径和基本信息
   - 在"字幕设置"标签页调整字幕样式、布局和动画效果
   - 点击"保存"按钮将设置保存到`config.jsx`文件

5. **生成视频**
   - 点击"开始"按钮，系统将自动处理素材并生成AE合成
   - 生成的合成会自动添加到渲染队列

## 目录结构说明
程序会自动创建以下目录结构：
```
工作目录/
├── 背景图层/
├── 封面图层/
├── 碟片图层/
├── 音乐/
├── 歌词/
└── 素材组/ (可选，用于分组管理素材)
```

## 注意事项
- 歌词文件需与对应的音乐文件同名，且放置在歌词目录中
- 如不放置歌词文件，将使用无字幕的模板布局
- 所有设置修改后需点击保存才能生效
- 生成的视频合成会自动按音乐文件名命名
- 生成滚动歌词的流程，因为会创建并编辑大量文本层，会产生较长时间卡顿，可尝试耐心等待
